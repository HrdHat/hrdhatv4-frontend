## Spacing Tokens

| Token                   | Value  | Description                                                          |
| ----------------------- | ------ | -------------------------------------------------------------------- |
| `--page-margin-mobile`  | `20px` | Distance of the paper from viewport edge on mobile and small screens |
| `--page-margin-desktop` | `50px` | Distance of the paper from viewport edge on desktop / large screens  |
